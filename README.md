# survey-land-backend-node

## Introduction
The **survey-land-backend-node** project is a Node.js and Express-based backend application designed to manage surveys and store files on Amazon S3. The project includes features such as file uploads, image conversion using Jimp, and application internationalization using i18n.

## Technologies Used
- **Node.js**: JavaScript runtime environment used on the server.
- **Express**: Web framework for building server-side applications.
- **Multer**: Middleware for handling file uploads.
- **Jimp**: Library for image processing and conversion.
- **AWS SDK v3**: For interacting with Amazon S3 services.
- **Passport**: For authentication and security mechanisms.
- **i18n**: For supporting multiple languages.
- **Helmet**: Enhances security by setting various HTTP headers.
- **Cors**: Enables Cross-Origin Resource Sharing configuration.
- **Morgan**: HTTP request logger middleware.
- **Dotenv**: Loads environment variables from a `.env` file.

## Requirements
- Node.js (version 14 or higher)
- npm
